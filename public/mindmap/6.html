▼ 🚦 PART 1~4. 신호기 분류·정위·확인거리
P1. 신호·전호·표지 구분: 신호=기관사 지시 / 전호=관계자 상호 / 표지=위치 표시
P2. 구조상: 완목식(주야 다름)·색등식·등열식(유도·중계 사용)
P2. 조작상: 수동·자동(폐색신호기)·반자동(장내·출발신호기)
P2. 주신호기 6종: 장출폐엄유입 / 종속신호기 3종: 원통중
P2. 임시신호기 3종: 서행예고·서행·서행해제 (중계신호기 포함 안 됨)
P3. 정위 암기: 장내·출발·입환·엄호 = 정지
P3. 정위 암기: 유도 = 소등(무현시) / 원방 = 주의
P3. 폐색신호기 정위: 복선=진행 / 단선=정지
P4. 확인거리: 장내·출발·폐색·엄호 = 600m 이상
P4. 확인거리: 입환·중계 = 200m / 유도 = 100m
P5. 특수신호 3종: 발뢰·발염·발광 (발폭 없음)
