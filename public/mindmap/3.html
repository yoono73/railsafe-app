▼ 🚂 PART 1~2. 운전규칙·폐색방식
P1. 동력차 연결 원칙: 맨 앞 / 예외 6가지(구원·제설·공사열차 등)
P1. 여객열차 화차 연결 금지 원칙 / 예외: 회송·특별사유
P1. 열차 안전 확보 3법: 폐색·열차제어장치·시계운전
P1. 퇴행운전 금지 원칙 / 예외: 선로·차량 고장, 공사·구원열차
P2. 상용폐색 4종: 자연차통 (자동·연동·차내신호·통표)
P2. 대용폐색 4종: 통지지지 (통신·지도통신·지도·지령식)
P2. 통표폐색: 단선 전용 / 해당 구간 통표 1개만 사용
P2. 지령식: 관제업무종사자 감시가능 + 통신장치 정상 → 시행
P2. 지도표 기입: 구간 양끝역명 + 발행일 + 사용열차번호
P2. 시계운전: 복선=격시·전령 / 단선=지도격시·전령
▼ 🚦 PART 3~4. 신호기·열차제어장치
P3. 주신호기 6종: 장출폐엄유입 (장내·출발·폐색·엄호·유도·입환)
P3. 종속신호기 3종: 원통중 (원방·통과·중계)
P3. 신호기 정위: 장내·출발·폐색·엄호·입환 = 정지
P3. 신호기 정위: 유도 = 소등(무현시) / 원방 = 주의
P3. 수신호 표시: 정지=양손X / 서행=한팔 좌우 / 진행=한팔 머리위
P4. ATS: 자동열차정지장치 → 경보 후 자동제동
P4. ATC: 자동열차제어장치 → 속도 초과 시 자동제어
P4. ATP: 자동열차방호장치 → ATS+ATC 기능 통합
P4. ATO: 자동열차운전장치 → 출발~정차 자동운전
